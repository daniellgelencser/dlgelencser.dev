# dlgelencser.dev

Interactive terminal-style CV website built with React and TypeScript.

## Overview

A single-page application presenting professional information through a conversational terminal interface. Users navigate content via interactive buttons with smooth typing animations.

## Tech Stack

- **React 18.3** - Component-based UI
- **TypeScript** - Type-safe development
- **Vite 6.0** - Fast dev server and optimized builds
- **CSS Modules** - Component-scoped styling

## Features

- Interactive conversation flow with typing effects
- Sections: About, Experience, Projects, Contact
- Responsive terminal interface
- Smooth scrolling and progressive content reveal
- Analytics integration ready (GoatCounter, Formbricks)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
